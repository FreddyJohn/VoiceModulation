<h1>VoiceModulation</h1>
<p>This is a lightweight digital audio workstation for android created fully from scratch</p>
<h2>Features</h2>
<ol>
  <li>14 time domain variable parameter modulations</li>
  <li>undo/redo</li>
  <li>project save</li>
  <li>edit operations, add/delete</li>
</ol>
